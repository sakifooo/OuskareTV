# **App Name**: كورة لايف - Koora Live (Pure HTML Edition)

## Core Features:

- Live Match Dashboard: A high-performance RTL match board built exclusively with semantic HTML and optimized CSS, displaying real-time scores, team flags, and commentator details for the Riyadh timezone.
- AI Tactical Analyst Tool: An integrated AI assistant that uses reasoning to provide match predictions and tactical summaries as a tool directly within the static HTML interface.
- Static Streaming Hub: Dedicated section for match broadcasts using standard HTML video embeds and placeholders for channels like beIN Sports and SSC.
- Zero-Framework News Portal: A lightweight football news feed utilizing standard HTML5 and CSS Grid for rapid loading and SEO-optimized Arabic content.
- Vanilla Data Sync: A streamlined mechanism to fetch and update live scores and league standings using vanilla JavaScript to maintain a pure HTML structure without external frameworks.

## Style Guidelines:

- Football Pitch Theme: Deep Charcoal (#121212) background with Stadium Green (#2E7D32) primary accents.
- Highlight Color: Vivid Lime (#C6FF00) for active match indicators and call-to-action elements.
- Arabic Typography: 'Cairo' font for high-impact headers and 'Tajawal' for clean, readable body text in RTL layout.
- Mobile-First RTL Grid: A responsive multi-pane layout optimized for Arabic reading directions without external UI frameworks.
- Minimalist monolinear icons representing football-specific elements like whistles, trophies, and stadiums.
- CSS-Only Pulse: Subtle pulsing green animations for live match statuses and smooth hover transitions on news cards.
